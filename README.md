# Autohmmer
Automated HMMER search tool to assist with deep homology searches. HMMER suite tools are excellent for finding divergent sequences.
This code requires a few input files. Please ensure you have a .fasta or .faa file from a Mulitple Sequence Alignment (MSA) before using this code. 
This code also requires a .fasta or .faa for the organism sequences you would like to search.
This code is written for use in terminal as a nano. Please save this code in terminal as a nano before use. 

This code was written with the HMMER 3.4 (Aug 2023) model, if code is not working please check https://hmmer.org/ before proceeding.

This code was written with the assistance of Claude Sonnet 4.6.
